# Health Status Prediction ML

A supervised machine learning project that predicts health status using health and lifestyle indicators.

## Models Used

- Logistic Regression
- KNN
- Random Forest
- Gradient Boosting
- Voting Classifier

## Best Model

Random Forest

- Accuracy: 93.82%
- Recall: 95.88%

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
